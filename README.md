## NOTICE
This repository contains the public FTC SDK for the DECODE (2025-2026) competition season.

## Welcome!
This GitHub repository contains the source code that is used to build an Android app to control a *FIRST* Tech Challenge competition robot.  To use this SDK, download/clone the entire project to your local computer.

## Requirements
To use this Android Studio project, you will need Android Studio Ladybug (2024.2) or later. If you are on one of Mr. Warren's computers, this should already be installed. If it is outdated or not installed, please ask Mr. Warren or a veteran team for help installing it from [https://developer.android.com/studio](https://developer.android.com/studio).
You can find more detailed installation directions here at this [Google Doc](https://docs.google.com/document/d/1PDQ6wohfdSlhczCxO9CT7zLG78vlwxjorjGLqmR30uE/edit?usp=sharing)
